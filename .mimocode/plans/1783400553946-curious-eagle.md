# Commit and Push Changes

## Goal
Run the git commit and push command that was already agreed upon.

## Command
```bash
git add . && git commit -m "feat: sidebar user info, URL-synced filters & pagination

- Add logged-in user profile (name, email) to sidebar footer with logout button and confirmation dialog
- Add useProfile hook that fetches /users/me and caches user data in auth store
- Extend auth store with user state (name, email) and setUser action
- Sync /users filters (search, isBanned, page) with URL search params
- Sync /announcements pagination with URL search params (?page=)
- Sync /audit-logs pagination with URL search params (?page=)
- Add reusable Pagination component to src/components/
- Add debounce (300ms) for users search input
- Reset page to 1 when search or status filter changes on /users
- Add loading states to login form buttons (isPending for login & verify)
- Switch useVerify to useAuthControllerVerifyAsAdmin for admin auth flow
- Await query invalidation in ban/unban hooks for proper sequencing" && git push
```

## Verification
- Confirm commit succeeds
- Confirm push succeeds
