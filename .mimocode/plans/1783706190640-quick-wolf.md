## Step 4: Build Reusable DataTable Component

Create a generic `DataTable` component in `src/components/data-table/` following the standard shadcn/ui pattern:

**File: `src/components/data-table/data-table.tsx`**

Core features:
- Generic `<DataTable<TData, TValue>>` component accepting columns and data
- Built on shadcn `Table` primitives (`Table`, `TableHeader`, `TableBody`, `TableRow`, `TableHead`, `TableCell`)
- Uses `@tanstack/react-table` for column definitions, sorting, filtering, row selection
- Integrates existing `Pagination` component (`src/components/pagination.tsx`)
- Loading skeleton state (table-specific, replacing the current pulse divs)
- Accepts optional `toolbar` slot for search/filters above the table

Also create:
- `src/components/data-table/data-table-column-header.tsx` — sortable column header with dropdown (sort asc/desc/hide)
- `src/components/data-table/data-table-toolbar.tsx` — optional toolbar wrapper for search input + filter controls
- `src/components/data-table/data-table-pagination.tsx` — wrapper around existing Pagination
- `src/components/data-table/data-table-skeleton.tsx` — table-shaped loading skeleton

**Key files to reference:**
- `src/components/ui/table.tsx` (shadcn table primitives)
- `src/components/pagination.tsx` (existing pagination)
- `src/components/ui/input.tsx` (search input)
- `src/components/ui/select.tsx` (filter dropdowns)
- `src/components/ui/badge.tsx` (status badges in tables)
